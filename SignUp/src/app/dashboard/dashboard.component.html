edit HOW TO CREATE TASK ?

  • Add task title.
  • Set a start date and time and due date and time.
  • Add Task Description.
  • Click on create and add more people.
  • You are good to go.

Thankyou For Reading.

edit HOW TO CREATE MEETINGS?

  • Add meeting title.
  • Enter place value.
  • Set a start date and time and due date and time.
  • Add meeting description.
  • Click on create and add more people
  • You are good to go.

Thankyou For Reading.

assignment HOW TO VIEW TASK?

1. You can use the cursor to click the Task View icon on the sidenav bar to run “Task View”.

2. In Task View, it will list all current windows you are using of the desktop on the top side, and you can select any to quickly open.
